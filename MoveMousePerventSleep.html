==========> ROS２インストール　<========== aptリポジトリの追加 sudo apt update && sudo apt install curl gnupg lsb-release sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg リポジトリをsource listに追加 echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(source /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null 動作テスト Terminal1: source /opt/ros/humble/setup.bash ros2 run demo_nodes_cpp talker Terminal2: source /opt/ros/humble/setup.bash ros2 run demo_nodes_py listener 環境設定 echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc source .bashrc ============================
